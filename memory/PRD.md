# Global Telecom - Product Requirements Document

## Project Overview
**Name:** Global Telecom Website
**Type:** Marketing/Landing Page
**Tech Stack:** Next.js 16, TypeScript, Tailwind CSS v4
**Repository:** https://github.com/PLP-N8n/Global

## Original Problem Statement
User wanted to modify the existing Global Telecom website (an electronics retailer in Panipat, Haryana) with:
1. Luxurious and clean design aesthetic
2. SEO-optimized layout rearrangement
3. Classic and elegant typography
4. Modern glassmorphism components

## User Personas
1. **Local Customers** - Residents of Panipat looking for electronics
2. **Tech Buyers** - People seeking mobile phones, TVs, accessories
3. **Value Seekers** - Customers wanting genuine products with warranty

## Core Requirements (Static)
- [x] Single-page marketing site
- [x] Bilingual support (English + Hindi)
- [x] Mobile-responsive design
- [x] WhatsApp & Call CTAs throughout
- [x] Trust indicators (Authorized Dealer, Genuine Products, Warranty)
- [x] SEO-optimized heading hierarchy

## What's Been Implemented (Feb 4, 2026)

### Design System - LUXURY Edition
- **Typography:** Playfair Display (headings) + Manrope (body) + Cinzel (accents)
- **Color Palette:** Midnight Navy (#0F172A), Champagne Gold (#D4AF37), Warm Off-Whites
- **Effects:** Glassmorphism cards, subtle shadows, gold accents

### Sections Implemented
1. **Hero** - Full-screen immersive with background image, glass card, CTAs
2. **Trust Banner** - 3 trust indicators (Genuine Products, Authorized Dealer, Warranty)
3. **About** - Local story with 15+ years stat, moved up for SEO
4. **Products** - Bento grid layout (Featured TV + 3 category cards)
5. **Brands** - 12 brand cards with hover effects
6. **Services** - 3 service cards + glassmorphism benefits section
7. **Contact** - Full contact info, WhatsApp CTAs, map placeholder
8. **Footer** - Quick links, categories, contact info

### Components
- Glassmorphism Card, Button variants (gold, whatsapp, outline, glass)
- Sticky header with scroll effect
- Mobile navigation with hamburger menu
- Floating WhatsApp button (mobile only)
- FadeIn animation wrapper

### Features
- SEO metadata with keywords
- Responsive design (desktop/tablet/mobile)
- Smooth scroll navigation
- Bilingual CTAs and content
